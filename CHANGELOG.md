## [1.0.0] - 2026-04-21

### Added

- Auto-suspend inactive tabs with configurable timeout (default: 30 minutes)
- Smart tab grouping by domain
- Save and restore named sessions with scroll positions
- Focus mode — close all tabs except current domain, restore later
- Memory usage dashboard with savings tracker
- Fuzzy tab search across all open and suspended tabs
- Whitelist domains from auto-suspension
- Preserve pinned tabs, audio-playing tabs, and tabs with unsaved forms
- Dark mode support (follows system preference)
- Keyboard shortcuts for common actions
- Chrome, Firefox, and Edge support (Manifest V3)
- Lightweight suspended tab placeholder page
- Extension badge showing suspended tab count
